**Автор:** [Вдовенков Глеб]  
**Дата создания:** Май 2026  
**Язык:** Python 3  
**GUI-фреймворк:** `tkinter` (входит в стандартную библиотеку Python)

## ?? Описание
`Movie Library` — настольное GUI-приложение для удобного хранения и управления личной коллекцией фильмов. Программа позволяет добавлять информацию о фильмах, просматривать список в табличном виде, фильтровать записи по жанру и году выпуска, а также сохранять и восстанавливать данные из файла в формате JSON. Все вводимые данные проходят автоматическую валидацию для исключения ошибок ввода.

## ? Функционал
- ? Добавление фильма с полями: `Название`, `Жанр`, `Год выпуска`, `Рейтинг`
- ? Валидация данных:
  - Год проверяется на корректность (должен быть целым числом)
  - Рейтинг проверяется на диапазон (строго от 0.0 до 10.0)
- ? Таблицы отображение коллекции с автопрокруткой
- ? Фильтрация:
  - По жанру (частичное совпадение, без учёта регистра)
  - По году (точное совпадение)
- ?? Сохранение коллекции в `movies.json` и автоматическая загрузка при запуске
- ?? Кнопка сброса фильтров

## ?? Требования
- Python 3.8 или выше
- Операционная система: Windows, macOS или Linux
- Сторонние зависимости не требуются (используются только стандартные модули `tkinter`, `json`, `os`)
